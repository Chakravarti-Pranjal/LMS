# LMS Frontend

### Setup instruction 

1. Clone the project

...
    git clone https://github.com/Chakravarti-Pranjal/LMS.git
...

2. Move into the directory

...
    cd LMS
...

3. install dependecies

...
    npm i
...

4. run the server

...
    npm run dev
...

# React + Vite
